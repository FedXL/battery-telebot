В общем прими эти поправки/ дополнения:
- Добавить дополнительный экран после нажатия на кнопку на экране выбор Покупателя или Продавца, с редактируемыми кнопками. Две кнопки на экране - Подтвердить, Назад.
- Добавить дополнительные кнопки на экране Меню Покупателя/ Продавца - правила Акции, Частые вопросы.
- Добавить Дополнительный экран с информацией правила Акции, кнопка назад.
- Добавить Дополнительный экран - часто задаваемые вопросы, кнопка назад.
- Добавить перед записью личных данных в профайл добавить экран где будет реплика со ссылкой на Политику конфиденциальности и Кнопки назад, Принять правила.
- Электронный адрес для организатора не обязателен - можно добавить кнопку Пропустить.
- При запросе личных данных на всех экранах отображать кнопку Назад для возможности вернуться на предыдущий экран для корректировки данных.
